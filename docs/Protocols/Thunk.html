Thunk

public protocol Thunk : Action

Thunk is a type of Action that allows for asynchronous operations. Thanks to DispatchFunction this action can dispatch other actions to applications store. Since Swift doesn’t allow for protocols to have Generic types without the use of assosiatedTypes, GetState function returns some object that conforms to StateProtocol which needs to be cast to your desired state type.
